# 🚀 AlgoMentor AI

AlgoMentor AI is an **AI-powered DSA (Data Structures & Algorithms) assistant** that helps users:

* Solve coding problems
* Understand concepts easily
* Get clean explanations like a mentor

---

## ✨ Features

* 🤖 Ask DSA questions in natural language
* 📚 Get simple and structured explanations
* 🕘 History tracking of previous questions
* 👤 User profile management
* 🎯 Clean and modern UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Vercel Serverless Functions
* **AI:** Gemini API
* **Deployment:** Vercel

---

```
AlgoMentor-AI/
```

---

## ⚙️ Setup & Run (Local)

> Note: AI will NOT work locally without API key setup

```bash
git clone https://github.com/itskeshavsharma/Algomentor-ai.git
cd Algomentor-ai
```

Open `index.html` in browser.

---

## 🚀 Deployment (Vercel)

1. Go to Vercel
2. Import GitHub repo
3. Add Environment Variable:

```
API_KEY=your_gemini_api_key
```

4. Click Deploy

---

## 🔐 Important

* Do NOT expose your API key in frontend
* Always use backend (`/api/ask.js`)
